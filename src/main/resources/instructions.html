How To Play Sudoku

General Sudoku Rules

  • The goal is to fill each 3 x 3 square with the numbers 1-9.
  • Each number can only appear once per row, per column, and per subsquare.
  • The game is won when the puzzle is fully completed.

How To Play

  • Use the mouse or the arrow keys to highlight the desired square.
  • Use the number keys or provided buttons to input a value into the square.
  • If it is a valid input, the number will appear in the highlighted square.
  • If it is not a valid input, nothing will change and the puzzle will remain the same.

Good Luck!
